# BBJ Landing Page (Frontend)

Repositori frontend utama untuk portal informasi **Berbagi Bites Jogja (BBJ)**.

## Deskripsi
Landing page ini berfungsi sebagai wajah digital BBJ untuk publik. Menyajikan informasi mengenai misi komunitas, statistik dampak, program unggulan, dan pintu masuk bagi calon donatur maupun relawan.

## Informasi Konten
- **Program:** Makan (Penyelamatan, Distribusi, Edukasi).
- **Dampak:** Statistik makanan yang diselamatkan, porsi dibagikan, dan mitra.
- **Ajakan Aksi:** Donasi, Daftar Relawan, Jadi Mitra.

## Teknologi
- Framework/Library: React/Next.js (sesuai folder structure).
- Styling: Vanilla CSS / Tailwind (sesuai folder structure).
